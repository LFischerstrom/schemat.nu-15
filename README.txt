=== INCLUDE LATEST COURSES AND GROUPS ===
1. Run updateIds.php. (Html-file will be downloaded. Json and js files will be created).
2. Next download process will use these updated groups and courses.

=== DOWNLOAD LATEST SCHEDULES ===
1. Open page script cronjob.html.
2. Make sure it never closes and dont shut down the computer.
3. All schedules will be downloaded once a day.

=== UPDATE STATS ===
1. Run cronjobs/updateStats.php
2. stats/stats.txt will be updated.
3. Visit Stats.php to display the updated stats.
